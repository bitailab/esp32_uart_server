# ESP32_UART_SERVER

这是一个串口服务器，基于 UART_ECHO 和 wifi_managed 开源项目修改得到，目前是个初级版本。它支持Web 的方式进行配网。

## 相关配置
代码在 ESP32-WROOM-32 的模组测试通过，实际使用中要注意 UART 和引脚的选择，在代码中进行修改。
